# Анализ инвестиций через T-Bank MCP

## Инструменты

| Тул | Что показывает |
|---|---|
| `get_data("invest_accounts")` | Инвест-счета (брокерские + InvestBox) |
| `invest_portfolio(broker_account_id, days)` | Статистика портфеля (сумма, P&L, доходность) |
| `invest_operations(broker_account_id, operation_type, limit)` | Операции (покупки/продажи/вводы) |
| `invest_securities(broker_account_id)` | Ценные бумаги в портфеле (акции/облигации/ETF) |
| `get_data("invest_offers")` | Предложения InvestBox |
| `get_data("invest_yield")` | Доходность продуктов InvestBox |
| `get_data("pension")` | Пенсионный профиль |
| `get_data("broker_margin")` | Маржинальные показатели |

## Флоу

### Шаг 1: Найти инвест-счета

1. `get_data("invest_accounts")` — получить список брокерских счетов.
2. Если счетов несколько — показать пользователю:
```
Брокерские счета:
  1. Основной (2243834454) — доступно 150 000 ₽
  2. ИИС (2243834455) — доступно 80 000 ₽
  3. InvestBox — доступно 25 000 ₽
```
3. Пусть выберет, или анализируй все.

### Шаг 2: Сбор данных по портфелю

Для каждого брокерского счёта:
1. `invest_portfolio(broker_account_id, days)` — статистика:
   - `totalAmount` — текущая стоимость портфеля
   - `expectedYield` — ожидаемая доходность
   - `expectedAverageYield` — средняя доходность
   - `marginEnabled` — маржинальность
   - `openedDate` — дата открытия

2. `invest_securities(broker_account_id)` — бумаги в портфеле:
   - акции, облигации, ETF
   - количество, цена покупки, текущая цена
   - P&L по каждой бумаге

3. `invest_operations(broker_account_id, "payOut", 50)` — выводы (последние 50)
   `invest_operations(broker_account_id, "payIn", 50)` — вводы
   `invest_operations(broker_account_id, "", 50)` — все операции (покупки/продажи)

### Шаг 3: Анализ (LLM-интеллект)

**Аллокация** — распределение по классам активов:
```
Акции:     105 000 ₽ (70%)
  - РФ:     80 000 ₽ (53%)
  - США:    25 000 ₽ (17%)
Облигации: 30 000 ₽ (20%)
ETF:       15 000 ₽ (10%)
Итого:     150 000 ₽
```

**P&L** — прибыль/убыток:
```
Общий P&L:     +12 500 ₽ (+9.1%)
  Лучший:     Яндекс +4 200 ₽ (+15%)
  Худший:     Сбербанк -1 800 ₽ (-6%)
  Нейтральные: Облигации +1 100 ₽ (+3.7%)
```

**Доходность vs период** — сравни с предыдущим месяцем:
- `invest_portfolio(broker_account_id, 30)` vs `invest_portfolio(broker_account_id, 90)`
- Покажи динамику: «За 30 дней +4.2%, за 90 дней +8.1%»

### Шаг 4: Рекомендации (LLM + данные)

1. **Ребалансировка**: «Акции 70% — агрессивно. Для твоего возраста рекомендуем 60/30/10 (акции/облигации/кэш). Продай часть акций, купи облигации.»
2. **Tax-loss harvesting**: «Сбербанк -1 800 ₽ — фиксируй убыток для налоговой (снижение НДФЛ).»
3. **ИИС**: «У тебя есть ИИС — убедись что внёс максимальную сумму (1 000 000 ₽/год) для налогового вычета.»
4. **Инвест-счета (InvestBox)**: `get_data("invest_offers")` + `get_data("invest_yield")` — предложения по доходности.
5. **Дивиденды**: посмотри операции — есть ли дивидендные выплаты.

**Не давай конкретных рекомендаций по покупке/продаже отдельных бумаг** — ты не лицензированный советник. Давай общие принципы (аллокация, диверсификация, налоги).

### Шаг 5: Отчёт

```
📈 Инвестиционный отчёт

Брокерский счёт: 2243834454
Стоимость портфеля: 150 000 ₽
Общий P&L: +12 500 ₽ (+9.1%)
Доходность (30 дней): +4.2%
Доходность (90 дней): +8.1%

Аллокация:
  Акции:     70%  (105 000 ₽)
  Облигации: 20%  (30 000 ₽)
  ETF:       10%  (15 000 ₽)

Топ-3 по P&L:
  1. Яндекс (YDEX):   +4 200 ₽ (+15%)
  2. ОФЗ 26238:       +1 100 ₽ (+3.7%)
  3. VTB Russia (VTBR):+800 ₽ (+5%)

Худший:
  - Сбербанк (SBER):  -1 800 ₽ (-6%)

Рекомендации:
  1. Ребалансировка: снизить акции до 60%
  2. Зафиксируй убыток по SBER для НДФЛ
  3. Проверь взнос по ИИС (1 млн ₽ до конца года)

⚠️ Не является индивидуальной инвестиционной рекомендацией.
```

## Правила

1. **P&L — в ₽ и %** — показывай обе метрики.
2. **Сравнение периодов** — 30 vs 90 дней (динамика).
3. **Без конкретных советов по бумагам** — только принципы (аллокация, диверсификация).
4. **ИИС** — проверяй взнос + тип вычета (A/B).
5. **Налоги** — напоминай про НДФЛ, фиксацию убытков, сроки уплаты.
6. **Денег не двигает** — аналитика только (нет переводов/покупок/продаж).
