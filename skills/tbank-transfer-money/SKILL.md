# Переводы через T-Bank MCP

## Инструменты

| Тул | Что делает |
|---|---|
| `list_accounts()` | Список счетов пользователя (найти источник) |
| `get_data("requisites")` | Реквизиты счёта (номер, БИК — для входящих) |
| `payment_commission(body)` | Предпросмотр комиссии (БЕЗ денег) |
| `transfer(amount, to_account, description)` | P2P перевод (HMAC-подпись внутри) |
| `pay(body)` | Сырой подписанный платёж (продвинутый) |

## Флоу

### Шаг 1: Определить тип перевода

| Пользователь сказал | Тип | Провайдер |
|---|---|---|
| «переведи на 79991234567» | P2P по телефону | `p2p-anybank` |
| «переведи на счёт 40817810100000001234» | Перевод по счёту | `tinkoff` (или другой банк) |
| «переведи по СБП» | СБП | `sbp` |
| «переведи между счетами» | Внутренний | `internal` |

Если не указан — уточни: «Перевод по телефону (P2P) или по номеру счёта?»

### Шаг 2: Найти счёт-источник

1. Вызови `list_accounts()` — показать счета с балансами.
2. Если у пользователя один счёт с деньгами — выбери его.
3. Если несколько — спроси: «С какого счёта: Current (2751₽) или Credit (доступно 50000₽)?»
4. Запомни `account_id` источника.

### Шаг 3: Проверить комиссию (ОБЯЗАТЕЛЬНО до перевода)

Вызови `payment_commission(body)` с `payParameters`:
```json
{
  "payParameters": {
    "account": "<account_id источника>",
    "moneyAmount": <сумма>,
    "currency": "RUB",
    "paymentType": "Payment",
    "provider": "p2p-anybank",
    "providerFields": {
      "pointerType": "ACCOUNT",
      "account": "<получатель>"
    }
  }
}
```
Получишь комиссию + описание. Покажи пользователю.

### Шаг 4: Подтверждение пользователя — ОБЯЗАТЕЛЬНО

Покажи ВСЕ данные перед переводом:
```
Перевод:
  Откуда: Current (2751 ₽)
  Куда: +7 999 123-45-67
  Сумма: 1500 ₽
  Комиссия: 0 ₽ (бесплатно)
  К зачислению: 1500 ₽

Подтверждаешь? (да/нет)
```

**НЕ переводи без явного подтверждения.** Если пользователь сказал «переведи» в исходной команде — это НЕ подтверждение, это запрос. Всегда показывай детали и жди «да».

### Шаг 5: Перевод

Вызови `transfer(amount, to_account, description)`:
- `amount` — сумма в рублях (число)
- `to_account` — номер телефона/счёта получателя
- `description` — комментарий (по запросу пользователя)

Для сложных случаев (не-P2P) — `pay(body)` с форматом `payParameters=...`.

### Шаг 6: Отчёт

```
✅ Перевод выполнен
  Сумма: 1500 ₽
  Получатель: +7 999 123-45-67
  Комиссия: 0 ₽
  Остаток на счёте: 1251 ₽
```

При ошибке — покажи код ошибки + предложи варианты.

## Правила

1. **Всегда проверяй комиссию** перед переводом — даже если бесплатная.
2. **Всегда подтверждай** с пользователем — сумма + получатель + комиссия.
3. **Сумма — число** (1500, не "1500 рублей").
4. **Получатель** — телефон в формате `+79991234567` или счёт.
5. **P2P по телефону** — дефолтный вариант.
6. **Не переводи с кредитки** без явного согласия (комиссия + проценты).
